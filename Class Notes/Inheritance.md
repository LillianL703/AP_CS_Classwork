##Inheritance: 
Inheritance defines a relationship between objects that share characteristics.

##Superclass:
A class that is inherited from a subclass.

##Subclass: 
It is created from an existing class, superclass, by absorbing its state and behavior and augmenting these with features unique to the new class.

##Extends:
The inheritance relationship between a subclass and a superclass is specified in the declaration of the subclass, using the keyword extends.

##Super():A subclass constructor can be implemented with a call to the super method, which invokes the superclass constructor.
